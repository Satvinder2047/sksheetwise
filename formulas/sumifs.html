Excel SUMIFS Function

Adds up values that meet multiple conditions simultaneously.

[ DISPLAY AD: RESPONSIVE BANNER ]

Syntax & Structure

=SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)

Unlike SUMIF, the range containing the values to sum (sum_range) is supplied first.

Interactive Examples

Example 1: Total Department Payroll
=SUMIFS(Payroll, Department, "HR", Month, "Jan")

Explanation: Sums the Payroll column only where the Department is "HR" and the Month is "Jan".

HR & Operations Applications

  • Totaling payroll cost by department, location, or pay period.
  • Summing overtime hours only for employees over a specific attendance threshold.
  • Building self-updating salary summary dashboards.
[ DISPLAY AD: RESPONSIVE CONTENT BANNER ]

Common Mistakes to Avoid

  • Argument Mismatch: SUMIFS has a reversed argument order compared to SUMIF. If you put sum_range last, Excel returns a formula error.
  • Mismatched Range Sizes: All criteria ranges must match the exact dimensions of your sum_range.
